# Práctica 2: Análisis de Estructuras Formales y Clausura de Kleene

## Descripción

Este proyecto corresponde a la implementación de una aplicación web que permite analizar y aplicar operaciones sobre lenguajes formales, enfocándose en la clausura de Kleene y otras operaciones fundamentales.

El sistema permite al usuario interactuar mediante una interfaz gráfica, seleccionando distintas operaciones y visualizando los resultados en tiempo real.

## Objetivos

- Generar cadenas a partir de un alfabeto definido.
- Verificar la pertenencia de una cadena a un lenguaje.
- Aplicar operaciones de unión y concatenación.
- Implementar la clausura de Kleene (L*) y Kleene Plus (L+).
- Analizar el crecimiento de un lenguaje.
- Integrar lógica de backend con una interfaz web.

## Tecnologías utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript
